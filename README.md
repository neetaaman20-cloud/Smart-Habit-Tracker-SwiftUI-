# Smart Habit Tracker (SwiftUI)

Smart Habit Tracker is a simple iOS application built using **Swift** and **SwiftUI** that helps users build daily habits and maintain streaks. The project focuses on clean architecture, readable code, and Apple-native design principles.

This repository is intended as a learning-focused, portfolio-ready Swift project.

---

## Features
- Create and manage daily habits
- Mark habits as completed
- Automatic daily streak tracking
- Local data persistence using UserDefaults
- Clean SwiftUI-based UI

---

## Tech Stack
- Swift
- SwiftUI
- MVVM-style architecture
- UserDefaults for persistence
- Xcode & iOS Simulator
